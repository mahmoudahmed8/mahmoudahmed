System management: The task of managing the system is one of the tasks of the operating system through which it manages the startup of the computer, as it makes sure that all the different elements of the computer are working properly, and once you make sure that the processes occur correctly, the user can start using the computer without any Problems.

Communication services: The communication services task is one of the tasks of the operating system that establishes connection to the Internet, as the operating system manages protocols in order to ensure the connection to the Internet, so that the user can browse the Internet, and the operating system manages the user's connections to networks that extend across the world, and ensure communication with networks Without any problems.

Security: This function provides protection against any malware that sometimes leads to data loss, system crashes, file tampering, or computer virus transmission.

Hardware services: One of the tasks of the operating system through which it manages computer parts related to the input and output operations such as printers, keyboard or DVD players.
